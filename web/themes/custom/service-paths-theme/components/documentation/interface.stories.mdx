# Storybook's Interface
## Sidebar

The sidebar is the most important navigation element in Storybook. It contains a menu
of the documentation pages as well as all components. The components are grouped by
[atomic design principles](?path=/story/documentation-component-structure--page#atomic-design-principles).

Some menu items are nested. This is indicated by a small triangle in front of the menu
item. Clicking on these menu items will expand the item and reveal the nested links.
![Storybook's sidebar](/documentation/sidebar.jpg)

## Search

At the top of the sidebar is a search box that allows you to search components by name.
![Storybook's search](/documentation/search.jpg)

## View options & shortcuts

Clicking on the 3 horizontal dots button will reveal an overlay menu with navigation options and options for
adjusting the current view. E.g., hiding the sidebar, toolbar or addons panel.
![Storybook's view options](/documentation/view-options.jpg)

## Toolbar

The toolbar is located at the top of Storybook. It includes options for changing the layout and switching
the language.
![Storybook's toolbar](/documentation/toolbar.jpg)

Using the language switcher you can switch the language of the documentation.
![Storybook's language switcher](/documentation/language-switcher.jpg)


## Addon panel

The addon panel sits at the bottom of Storybook. Make sure that it's enabled in
[view options & shortcuts](#view-options--shortcuts). The panel contains tabs for automated accessibility
checks and viewing the HTML source among others.
![Storybook's addon panel](/documentation/addon-panel.jpg)

# Storybookin käyttöliittymä

## Sivupalkki

Sivupalkki on tärkein valikko Storybookissa. Sen avulla voit selata komponentteja ja dokumentaatiota.
Komponentit on järjestelty [atomic design -periaatteiden]
(?path=/story/documentation-component-structure--page#atomic-design-principles) mukaan.
Joillakin valikon elementeillä on alavalikko, jonka osoittimena on pieni kolmio. Alavalikko avautuu
elementtiä klikkaamalla.

![Storybookin sivupalkki](/documentation/sidebar.jpg)

## Haku

Sivuvalikon ylälaidassa on hakukenttä, jonka avulla komponentteja voi etsiä nimen perusteella.

![Storybookin haku](/documentation/search.jpg)

## Näkymän hallitseminen ja näppäinkomennot

Sivupalkin yläreunassa olevat kolme pistettä avaavat valikon, josta voi valita, mitä elementtejä haluaa sisällyttää
Storybook-näkymään (esim. sivupalkki, tyokalupalkki ja liitännäispaneli). Valikosta näkee myös näppäinkomennot näkymän
hallitsemiseen ja komponenttien välillä liikkumiseen.

![Storybookin näkymävalikko](/documentation/view-options.jpg)

## Työkalupalkki

Storybook-näkymän ylälaidassa on työkalupalkki, jossa on mm. työkalut saavutettavuuden testaamiseen. Työkalupalkki
sisältää myös mahdollisuuden vaihtaa dokumentaation kieltä, siirtyä koko näytön näkymään ja avata komponentti
erillisellä välilehdellä.

![Storybookin työkalupalkki](/documentation/toolbar.jpg)

Using the language switcher you can switch the language of the documentation.
![Storybook's language switcher](/documentation/language-switcher.jpg)


## Liitännäispaneli

Storybook-näkymän alalaidassa on littännäispaneli. Sen voi sulkea painamalla sen oikeassa ylälaidassa näkyvää ristiä,
ja sen saa näkyviin valitsemalla _Show addons_ sivupalkin yläreunassa olevasta [valikosta, jossa näkymää voi hallita]
(#view-options--shortcuts).

Panelissa näytetään automaattiset saavutettavuusvarmistukset, ja sen avulla voi nähdä myös komponentin HTML:n.

![Storybookin liitännäispaneli](/documentation/addon-panel.jpg)
